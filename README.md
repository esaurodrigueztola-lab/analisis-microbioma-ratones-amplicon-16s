# Análisis de Microbioma Intestinal de Ratones (16S rRNA)
## Descripción
Repositorio con el código R para el análisis bioinformático del microbioma intestinal de ratones mediante secuenciación 16S rRNA.

## Cómo usar
1. Clonar el repositorio: `git clone https://github.com/TU_USUARIO/analisis-microbioma-ratones-16s.git`
2. Abrir el script `analisis_principal.R` en RStudio.
3. Instalar los paquetes según las instrucciones al inicio del script.
4. Descargar las bases de datos de referencia SILVA v138 y colocarlas en una carpeta `references/`.
5. Ejecutar el script paso a paso.

## Requisitos
- R (versión 3.1 o superior)
- Paquetes: dada2, phyloseq, ggplot2, DECIPHER, etc.
